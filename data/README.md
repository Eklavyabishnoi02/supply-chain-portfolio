# Data

Project data. **Raw** files stay local; **samples** are committed for reproducibility.

## Folders
- `raw/` — full downloads (ignored by git)
- `processed/` — intermediate outputs (ignored by git)
- `sample/` — small CSVs used by notebooks (committed)
